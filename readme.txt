Create stunning blog archive pages with Elementor using Grid, Masonry, and Portrait layouts—fully customizable styles and controls.


** Description **

Build stunning, fully customizable archive layouts with Elementor Archive Studio—powerful, flexible, and no code needed.

This addon includes 4 professionally designed archive layouts:

- Grid Layout

- Masonry Layout

- Portrait Layout

- Portrait Card Layout

Each layout comes with separate layout controllers and styling options, giving users total control over how their archive pages look.
Change spacing, typography, image size, metadata, pagination styles, and much more.

This plugin also supports advanced pagination styles, letting users define how archive navigation appears—classic pagination, load more, infinite scroll, numbers, next/prev buttons, etc.


** Key Features ** 

✔️ 4 Modern Archive Layouts

✔️ Separate Controller for Each Layout

✔️ Full Style Customization (Typography, Colors, Spacing, etc.)

✔️ Advanced Pagination Options

✔️ Works with Any Theme

✔️ Optimized for Elementor & Elementor Pro

✔️ Fast Loading & SEO-Friendly Output

✔️ Lightweight & No Unnecessary Scripts

✔️ Developer-friendly structure for adding more layouts in future.

Whether you're a designer or developer, you can easily create beautiful archive pages that perfectly match your brand.


** Features & Benefits **

** - Dynamic Layout Selection **
Users can choose any archive layout, and the plugin automatically applies it on the frontend.

** - Global Customization Support ** 
Customize archive elements such as:

- Featured images

- Titles

- Meta info

- Excerpt

- Read More button

- Spacing & padding

- Border, radius, shadow

** - Extendable Layout System ** 
Developers can add additional layouts with separate controllers.

** - Elementor-Native Controls ** 
All settings use Elementor’s built-in Control Manager and styling structure.


** Frequently Asked Questions ** 

1. Does this plugin work with Elementor Free?

Yes! It works with both Elementor Free and Elementor Pro.

2. Can I create custom archive layouts for CPTs (Custom Post Types)?

Yes. All custom post type archives are fully supported.

3. Is the plugin lightweight?

Absolutely. It loads only required CSS/JS per layout.

4. Can I add my own custom archive layout?

Yes. The plugin is coded in a developer-friendly way so additional layouts can be added easily.

5. Does it support WooCommerce product archives?

Yes, WooCommerce archive layouts are supported.
